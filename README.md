# Portfolio Muhammad Irfan

## Deskripsi
Website portofolio pribadi untuk menampilkan data diri, hobi, pemain favorit, dan form kontak.

## Teknologi
- HTML 5 (Semantic)
- CSS 3 (Responsive Design, Dark Mode)
- Vanilla JavaScript (ES6+)

## Fitur
✅ Dark Mode Toggle
✅ Smooth Scroll Navigation
✅ Fade In Animation
✅ Back to Top Button
✅ Form Validation
✅ Responsive Mobile & Desktop
✅ Lazy Load Images

## Cara Menjalankan
1. Clone repository
2. Buka file `index2.html` di browser
3. Atau deploy ke Vercel/Netlify

## Browser Support
- Chrome, Firefox, Safari, Edge (terbaru)
